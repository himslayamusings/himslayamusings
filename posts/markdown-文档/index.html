Markdown 文档

简介

Markdown 是一种轻量级的标记语言，易于阅读，也 易于编写。它广泛应用于各种文档写作、博客平台、笔记工具等。

主要特性

  1. 简单易学
  2. 支持多种格式
  3. 跨平台兼容

在 Hugo 中，Markdown 文件（特别是 .md 文件）的头部配置部分被称为 Front Matter。Front Matter 使用 YAML、TOML 或 JSON 格式来定义文章的元数据。以下是一些常见的 Front Matter 参数：

title: “文章标题” date: 2024-08-20 draft: false summary: “这是文章的摘要。” author: “作者名字” type: “post” slug: “custom-url-slug” # 自定义 URL 的最后一部分 categories: [“分类1”, “分类2”] tags: [“标签1”, “标签2”] keywords: [“关键字1”, “关键字2”] description: “搜索引擎优化(SEO)的描述”

SEO 相关

title: “SEO 友好的文章标题” description: “适合搜索引擎的描述” keywords: [“SEO关键字1”, “SEO关键字2”]

布局和显示相关

layout: “custom-layout” # 使用自定义布局 aliases: ["/old-url/"] # 为旧 URL 添加重定向 showDate: true # 是否显示发布日期 hideComments: true # 是否隐藏评论 disableShare: true # 是否禁用分享按钮

文档内容和结构

series: [“系列文章1”, “系列文章2”] # 将文章归入系列 weight: 1 # 用于排序的权重值，较小的值会显示在较前面

文档图片和媒体

featured_image: “images/featured.jpg” # 特色图片 thumbnail: “images/thumb.jpg” # 缩略图 cover: “images/cover.jpg” # 封面图片 images:

  • “images/img1.jpg”
  • “images/img2.jpg”

自定义参数

customParam: “自定义参数值” # 你可以自定义任何参数，并在模板中使用它

多语言支持

title: “文章标题” date: 2024-08-20 draft: false

[languages] [languages.en] title: “English Title” description: “Description in English” [languages.zh] title: “中文标题” description: “中文描述”

其他常见参数

readingTime: true # 是否显示阅读时间 toc: true # 是否显示目录 lastmod: 2024-08-21 # 最后修改日期

这些参数可以在不同主题中有不同的使用方法，根据你使用的主题，有些参数可能会有特定的用途或表现形式。你可以根据需要在 Front Matter 中添加或删除参数，以控制文章的显示和功能。

工具和资源

Markdown 编辑器

  • Typora：所见即所得的 Markdown 编辑器，实时预览。
  • Visual Studio Code：强大的代码编辑器，支持 Markdown 语法高亮和预览。
  • Obsidian：适合知识管理的 Markdown 编辑器。

在线 Markdown 编辑器

  • Dillinger：Dillinger 是一个在线 Markdown 编辑器，可以实时预览和导出文档。
  • StackEdit：StackEdit 是另一个功能强大的在线 Markdown 编辑器，支持多种导出格式。

总结

Markdown 是一种简单、直观的文本格式化工具，适合用于文档编写、博客文章、笔记记录等多种场景。通过掌握上述语法和技巧，你可以轻松地编写出结构清晰、格式美观的文档。
